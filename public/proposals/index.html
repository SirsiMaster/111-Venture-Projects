Statement of Work (SOW)

Executive Summary

This Statement of Work documents the refined implementation plan for FinalWishes. It solidifies the decision to self-host key infrastructure (DocuSeal) and leverage the latest AI models (Gemini 3.0) to maximize efficiency and reduce operating costs.

The Reality: Most estate settlement is still paper-based. There are no unified government APIs. Financial institutions have proprietary or non-existent programmatic interfaces. Court systems vary by county, not just state.

The Innovation: Where connectors don't exist, we BUILD THEM. We become the benchmark for state adoption of automated estate settlement.

1. Strategic Approach

The $95K/5-month scope includes:

  • Full automation for states WITH e-filing (MD, IL, MN)
  • Form library covering ~45 probate forms across 3 states
  • Complete user guidance through every step (manual or automated)
  • Self-hosted DocuSeal for e-signatures (Postgres compatible, Cost saving vs DocuSign)

4. Technical Stack & Architecture

4.1 E-Signature (DocuSeal)

Decision: Use DocuSeal (Self-Hosted).

  • Architecture Synergy: DocuSeal supports PostgreSQL (Cloud SQL).
  • Control: Data stays in our infrastructure (Cloud SQL/Storage).
  • Cost: Fixed infrastructure cost (~$50/mo) vs per-envelope pricing.

4.2 Document Processing

Decision: Google Document AI (OCR, entity extraction).

4.3 Certified Mail

Decision: Lob (Address verification, certified mail tracking).

4.4 Data Architecture (Multi-Tenant)

Each estate's data must be completely isolated. We employ a hybrid architecture:

  • Firestore (NoSQL): User profiles, estate metadata, task tracking.
  • Cloud SQL (PostgreSQL): PII Vault (SSNs, Account Numbers) with row-level security.
  • Cloud Storage: Encrypted document vault with client-side AES-256-GCM.

5.2 Full Scope for $95K / 5 Months

INCLUDED IN INITIAL RELEASE:

  1. Core Platform: Auth, Estate Management, Vault, Beneficiary Tracking.
  2. Guidance System (The Shepherd): State checklists (5 states), Form library, Task tracking.
  3. Document Generation: Notification letters, Court forms (PDF), Certified mail.
  4. Full Integrations: Plaid (Account discovery), DocuSeal (Self-hosted), Lob, Vertex AI (Gemini 3.0).
  5. Mobile Apps: React Native iOS + Android with document capture.

6. Final Decision

FinalWishes will be a complete estate settlement platform that builds connectors where none exist.

The competitive moat is:

  1. State-specific expertise (5 states at launch)
  2. Form template library (~75 forms)
  3. Connector strategy (we build what government doesn't provide)
  4. Progress tracking (visibility into 18-month process)
  5. Secure document vault (zero-knowledge)

Can we build a functional estate settlement platform for $95K in 5 months? YES.
