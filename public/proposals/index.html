2. Architecture Decisions

Key decisions summarized below. Full justifications available in Architecture Decision Records.

2.1 Security: SOC 2 + AES-256 (Not FIPS 140-3)

Decision: Implement industry-standard SOC 2 security, NOT federal FIPS 140-3.

Industry Reality - Who Doesn't Use FIPS

CompanyWhat They HandleSecurity Standard
Betterment$40B+ in assetsSOC 2, standard encryption
Wealthfront$30B+ in assetsSOC 2, standard encryption
Trust & WillEstate documentsBasic cloud encryption
1PasswordPasswords, secretsAES-256, custom derivation
StripePayment card dataSOC 2, HSM for cards only
PlaidBank credentialsSOC 2, no FIPS

FIPS 140-3 cost impact: +$50,000-100,000/year, +3-4 months. Customer value: Zero.

2.2 Cloud KMS (Not HSM)

Decision: Use Google Cloud KMS software keys, NOT Hardware Security Modules.

Key ManagementMonthly CostAnnual Cost
Cloud HSM$1,200+$14,400+
Cloud KMS (software)~$10-50~$120-600
Savings$1,150/mo$13,800/yr

HSM is used by crypto exchanges and banking core systems. Consumer fintech uses software KMS.
