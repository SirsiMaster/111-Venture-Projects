2.2 Security Standard: SOC 2 + AES-256

Decision: Implement industry-standard SOC 2 security, NOT federal FIPS 140-3.

Industry Reality - Who Doesn't Use FIPS

Company What They Handle Security Standard
Betterment $40B+ in assets SOC 2, standard encryption
Wealthfront $30B+ in assets SOC 2, standard encryption
Trust & Will Estate documents Basic cloud encryption
1Password Passwords, secrets AES-256, custom derivation
Stripe Payment card data SOC 2, HSM for cards only
Plaid Bank credentials SOC 2, no FIPS

FIPS 140-3 cost impact: +$50,000-100,000/year, +3-4 months. Customer value: Zero.

2.2 Cloud KMS (Not HSM)

Decision: Use Google Cloud KMS software keys, NOT Hardware Security Modules.

Key Management Monthly Cost Annual Cost
Cloud HSM $1,200+ $14,400+
Cloud KMS (software) ~$10-50 ~$120-600
Savings $1,150/mo $13,800/yr

HSM is used by crypto exchanges and banking core systems. Consumer fintech uses software KMS.

2.3 Go Backend (Not Rust)

Factor Rust Go Winner
AI code generation Good Excellent Go
Firebase Admin SDK Community Official Go
Build times 2-5 min 10-30 sec Go
Cloud Run cold start ~100ms ~50ms Go
Future hiring Difficult Easy Go
